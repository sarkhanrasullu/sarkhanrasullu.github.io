Sıfırdan Sona Java ( 6 aylıq tam kurs )

YouTube-da iki qrupuma (Awesome Qrupu və SweetHome Qrupu) keçdiyim bütün dərslərin video qeydlərini yükləmişəm. 6 ay boyunca keçirilmiş bütün dərslərin video qeydlərinin hər biri YouTube kanalımda var. Standart Edition, Desktop Programlaşdırma, Enterprise Edition, Spring Framework, Spring Data, Spring MVC, Spring Security, JPA, Hibernate, Restful API, Soap API, Multithreading, TCP də daxil olmaqla. Kurs müddətində real layihə işlənir. Desktop Proqramlaşdırma, Web Proqramlaşdırma üzrə real layihələr edilir. Design Patternlər, Best Practicelər keçilir. Məqsəd sizi iş mühitinə tam hazırlamaqdır. Hər bir qrupun tələbələri fərqli şirkətlərdə proqramçı olaraq çalışırlar. Siz də bacara bilərsiniz. Sadəcə diqqətlə videolara baxın və suallarınız olduqca soruşun. Uğurlar!

Hansı qrupun dərslərinə baxmalıyıq?

Ümumilikdə Awesome və Sweethome qrupu eynidir, hər iki qrup sıfırdan Javaya hazırlaşmaq istəyənlər üçün nəzərdə tutulub və hər iki qrupun dərslərini bitirən tələbələr fərqli şirkətlərdə çalışırlar, lakin Awesome qrupunun dərsləri daha səlis və axıcı izah edilib, bəzi məqamlar daha dərin izah edilib və bəzi əlavə dərslər var. Məsələn Threadler həm Awesome, həm də SweetHome qrupunda keçilir, lakin Awesome qrupunda daha geniş izah edilir. Bəzi tələbələr üçün Awesome qrupunun dərsləri, bəzi tələbələr üçün isə SweetHome qrupunun dərsləri rahat gəlir. Mən Awesome qrupunun dərslərini məsləhət görürəm.

Uğur Qazana bilmək üçün dərslərə necə baxmalıyıq?

Bütün dərslərə baxarkən köklənin ancaq indi baxdığınız videoya və onu başa düşməyə çalışın. Video ətrafında yaranacaq heç bir sual sizi narahat etməsin. Əsas indiki dərsdə izah ediləni başa düşməkdir. Niyə? Keçdiklərimiz OOP və OOP-nin real layihələrdə tətbiqidir. Həm console, həm desktop, həm də webdə. OOP programlaşdırmada konstutusiya kimi bir anlayışdır. Sizdə yarana biləcək bütün suallar əslində OOP-nin hansısa mövzusudur və zatən biz dərsdə bütün OOP mövzularını keçirik və sizin sualınızın cavabı növbəti dərslərdə mütləq çıxacaq. Ona görə də indiki dərsə köklənin, bu dərsi anladınızsa əlavə sualları boş verin, keçin növbəti dərsə. Mövzuların hər biri keçilir və dəfələrlə çeynənilir. Əgər cari dərsdə 2+2 keçilirsə və siz bunu anladınızsa deməli sizdə alındı. Amma məsələn ay müəllim bəs 2-2 necə olur, 2*2 necə olur və s. bunları düşünməyin. Gələn dərslərsə çıxacaq qabağınıza. Dərslərə baxarkən aşağıdakılara əməl etməyə çalışın:
  1. Videolarda nə necə edilirse olduğu kimi tətbiq edin. Birə bir eynisini.
  2. Əşşi nə olacaq ki: qoy folder adı başqa olsun, qoy class adı belə olsun və s. kimi dəyişiklilər etməyin. Videoda necədirsə o qaydada edin.
  3. Sual soruşmazdan öncə baxın görün bu videoyaya mən yaxşı mı baxdım. Bəlkə sualının cavabı videoda var. 90% hallarda sualınızın cavabı videodadır. Sual soruşmağa tələsməyin
  4. Dərsə baxanda mütləq kod yazın. Əşşi mən baxdım başa düşdüm kod yazmıram ehtiyyac yoxdur. Belə etməyin. Çünki çox sadə məntiqlə kod yaza bilmək üçün kod yazmaq lazımdır :)
  5. Kodları qətiyyən əzbərləməyə çalışmayın və gözü yumulu videoya baxmadan kod yazmaq xəyalına qapılmayın. Bu yavaş yavaş videolara baxaraq, video ilə birlikdə kod yazaraq zamanla olacaq. Düşünmə qabiliyyətiniz, algorithm qurma qabiliyyətiniz dərslərə baxıb tətbiq etdikcə alınacaq. Bəzilərində 5, bəzilərində 8 dərsdən sonra artıq öyrəndiyini düşünərək yazma qabiliyyəti formalaşır, lakin yenə də nələrisə xatırlamaq üçün köhnə kodlara və ya videolara baxma ehtiyyacınız olacaq, bu normaldır. Əsas məsələ yazdığını anlamaqdır və kodu oxuya bilməkdir. 3 aylıq dərsdən sonra artıq daha səlis kod yazacaqsınız.
  6. Dərsə baxanda dəftərə not alın. İndi açıb komputerə not alacaqsınız. Halbuki mən dəftərə not alın dedim :). Etməyin belə. Nəyi necə deyirəmsə elə edin. Formuladan qırağa çıxmayın.
